Collapsibles are great for when you have a lot of content, but only want to show part of it unless the user wants to see more.

Basic Collapsible

To make a collapsible element, add the .collapse class to a collapsible element (such as a div); this is the content that will appear when the collapsible button it clicked.

Add the data-toggle="collapse" attribute to an < a > or < button > element

Then add the data-target="#id" attribute to connect the button with the collapsible content (for < a > element, put the #id in the href attribute instead):
