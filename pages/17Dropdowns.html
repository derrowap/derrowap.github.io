A dropdown is a menu that allows a user to choose one value form a predefined list.

Basic Dropdown

A .dropdown class on a < div > element indicates a dropdown menu.

Use a button or link with .dropdown-toggle class and data-toggle="dropdown" attribute to open the dropdown menu.

The .caret class is the down arrow icon indicating the button is a dropdown.
