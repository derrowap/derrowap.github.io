A progress bar can show a user how far along he/she is in a process.

Basic Progress Bar

Add a .progress class to a < div > element to create a progress bar. Not supported in Internet Explorer 9 and earlier because they use CSS3 effects instead. Use the aria attributes to help users who use screen readers:

70% Complete

Progress Bar With Label

Remove the .sr-only class from the < span > element to show visible percentage:

70% Complete

Colored Progress Bars

Contextual classes progress bars can use: .progress-bar-success, .progress-bar-info, .progress-bar-warning, .progress-bar-danger.

40% Complete (success)
50% Complete (info)
60% Complete (warning)
70% Complete (danger)
