Form Control States

  • INPUT FOCUS: outline of input is removed and box-shadow is applied on focus.
  • DISABLED INPUTS: Add a disabled attribute to disable an input field.
  • DISABLED FIELDSETS: Add a disabled attribute to a fieldset to disable all controls within it.
  • READONLY INPUTS: Add a readonly attribute to an input to prevent user input.
  • VALIDATION STATES: Give a validation style to input by adding .has-warning, .has-error, or .has-success to parent element.
  • ICONS: Add feedback icons with .has-feedback class and an icon.
  • HIDDEN LABELS: Add .sr-only class on non-visible labels
